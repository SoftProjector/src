3.1.2 ФорматSong Format

При добавлении или редактировании песен,ОЧЕНЬ ВАЖНО соблюдать заданный формат. При несоблюдении формата, песня может быть выведена на экран неправильно, либо может зависнуть программа.

Название куплета.
Каждый куплет должен иметь название. Софт Проектор поддерживает три варианта названий для куплета: Куплет, Припев и Слайд (см. ниже). Название куплета может быть введено кириллицей или латиницей и должно начинаться с большой буквы r Russian and must start from a beginning of a new line and must be capitalized.  After the stanza title (on the same line), a stanza number or any text can be entered.

Verse Block. Is shown only once, and refrain block will follow automatically if one exists.
Verse, &Verse
Куплет, &Куплет
Refrain Block. Will be automatically shown after each verse block and will not be shown after slide block. If there is more than one refrain block, then the second refrain block will replace the first one when it gets to it.
Chorus, &Chorus, Refrain, &Refrain,
Припев, &Припев
Slide Block. This stanza block is for stanzas that is other than verse or refrain. It will be shown only once and refrain block will not follow automatically after. If refrain block is needed after slide block, please add it manual.
Slide, Insert, Intro, Ending,
Вставка, Вступление, Окончание

Special Stanza Blocks
Verse block and refrain block, each have special continuing block. Sometime a verse or refrain is too large to put it into a one slide, adding & to next verse/refrain block will solve the problem. An "&" sing in front of verse/refrain title means that it is part of the block above it and the two block will act as one. See examples below.

Some Examples
Example 1 - as Edited.Куплет1, Припев, Куплет2, Куплет3.
Example 1 - Shown.Куплет1, Припев, Куплет2,Припев, Куплет3, Припев.
Example 2 - as Edited.Куплет1, Припев, Куплет2, Вставка, Куплет3.
Example 2 - Shown.Куплет1, Припев, Куплет2,Припев, Вставка,Куплет3, Припев.
Example 3 - as Edited.Куплет1, Припев1, Куплет2, Куплет3, Припев2.
Example 3 - Shown.Куплет1, Припев1, Куплет2,Припев1, Куплет3, Припев2.
Example 4 - as Edited.Куплет1a, &Куплет1b, Припев, Куплет2a, &Куплет2b, Куплет3a, &Куплет3b.
Example 4 - Shown.Куплет1a, Куплет1b, Припев, Куплет2a, Куплет2b, Припев, Куплет3a, Куплет3b, Припев.
Example 5 - as Edited.Куплет1a, &Куплет1b, Припев-a, &Припев-b, Куплет2a, &Куплет2b, Куплет3a, &Куплет3b.
Example 5 - Shown.Куплет1a, Куплет1b, Припев-a, Припев-b, Куплет2a, Куплет2b, Припев-a, Припев-b, Куплет3a, Куплет3b, Припев-a, Припев-b.
