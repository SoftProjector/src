Managing Songbooks

softProjector allows you to have as many songbooks as you would like to have. You are allowded to import, edit, export, delete a songbook.

To manage Bibles:
1. Edit>Manage Database. This will open a dialog where managing will take plase. Shortcut: Crtl+M
2. Select "Songbooks" tab.


Import:This will import a songbook.
Edit:A dialog box will apeare where you can change the name or title of the songbook.
Export: Esport a songbook from the database that you want to share with someone else.
Delete. Delete a songbook that you do not want to use any more. You will not be allowed to delete any more songbooks if only one songbook is left.
