Managing Bibles

softProjector allows you to have as many Bible version as you would like to have. You are allowded to import, edit, export, delete a Bible.

To manage Bibles:
1. Edit>Manage Database. This will open a dialog where managing will take plase. Shortcut: Crtl+M
2. Select "Bibles" tab.


Import:This will import a Bible.
Edit:A dialog box will apeare where you can change the name or title of the Bible.
Export: Esport a Bible from the database that you want to share with someone else.
Delete. Delete a Bible that you do not want to use any more. You will not be allowed to delete any more Bibles if only one Bible is left.
