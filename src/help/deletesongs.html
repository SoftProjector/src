Deleting Songs

 

To delete a song:
You will be able to delete a song that is currently selected.
1. Select a song that you want to delete.see showing songs on how to select a song.
2. Edit>Delete Current Song.A warning dialog box will pop up asking you if you really want to delet that song, click "Yes" if you really want to do so or click "No" if you do not want to delete that song. Shortcut: Crtl+D
