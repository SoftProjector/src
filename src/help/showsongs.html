To show a song:
1. Select a songbook. You may keep songbook drop box stay at all songbook and use filter box to help find the song. In order to use the song number only entry filed, a specific songbook needs to be seleted.
2. Select a song. To select a song, you may just select a song from song title table. If a spcific songbook is selected, you may enter the number in the song number entry field or in the filter filed. You may also select a song by entering text in the filter entry box
3. Click "Go Live" button After you have selected a song, cliking "Go Live" will show show currently selected verse/refrain in the song preview list. First item in the song preview list is alway automatically selected when a song is seleted. Alternatively you may double click any verse/chorus from the preview list and it will send current song to be shown and double clicked item as current item.

Searching for a song:
To search for a song, enter a any part of the word in the filter entry box. This will filter all song titles that contains entered charecters. You can enter either number of the song of any part of the song title.
"Match beginning" check box.If checked, softProjector will filter out only those item that start with the text entered in the filter entry box.
"Exact match" check box. If checked, softProjector will filter out only those items that exactly match the text in the filter entry box. This method is practical when fitering song numbers.

Show list tips:
When a verse/chorus is sent to be shown, the show list list gets populated with current song and current verse/corus is selected and displayed on the screen.
To show next verse/chorus, just use down or up keys to select a verse/chorus to be shown. When an item in show list is changed, the selected verse/chorus will be shown.
"Hide" button.Use this button to clear all text from the screen.
"Show" button. Use this button to show currently selected verese/chorus in the show list. Alternatively, you may just double click the verse/chorus and it wil be shown.

Playlist:
softProjector allows you to add song into a playlist until you either remove them from playlist or exit the program
Adding a song to playlist.To add a song to playlist 1) select a song from song table that you want to add to playlist 2) click "Add to playlist" button. You may also double click on a song from song table and it will add that song to playlist.
Removing a song from plalist.To remove a song from playlist 1) select a song from playlist table 2) click "Remove from playlist" button.
To show a song that is in playlist. 1) Select a song in the playlist 2) click "Go Live" button. You may also just double click on the song in the playlist and it will show that song.
