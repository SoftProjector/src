Song format
To add a new songbook

To add a new song:
1. Select "Songs" tab. You will be allowed to add a song only when "Songs" tab in selected.
2. Edit>New Song. Shortcut: Crtl+N
3. Select Songbook.Select a songbook that you want to add a new song to. If you do not want to add to any existing songbooks, you may add a new songbook. See also To add a songbook.
3. Enter song number.softProjector will automatically enter a next song number. If for any reason you do not want to use that number, you may change it.
4. Enter song title. You must have a song title for all song. All none alphanumeric charecters will be removed from the title. This is done so that song searching/filtering made esier.
5. Enter author of the word for this song. "Words By" entry field.
6. Enter authof of the music for this song. "Music By" entry field.
7. Enter tune.This is the tune/key/тональность that this song is sung in.
8. Select Category. optional. Select closed caterogry that this song falls into.
9. Enter song text. Song text must be properly formated in order not to crash the program and to show the song properly. See Song Format below.
10. Click "Save" button.If you do not want to save changes, click "Cancel" button

Song format:
When entering a song or editing a song, make sure that you keep proper song format. If the song format will not be kept, it may not show the song properly and might cause program crash.
Stanza Title. Each stanza must have a stanza title. You have the option of three stanza titles (currently only in Russian) Куплет. Припев and Вставка. After the stanza title (on the same line), you can have number or any text.
Куплет.(Verse) Is shown only once, and Припев will follow Куплет automaticallyif there it exists.
Припев.(Chorus/Refrain) Will be automatically shown after each Куплет and will not be shown after Вставка. If there is more than one Припев, then the second Припев will replace the first one when it gets to it.
Вставка.(Insert) Will be shown only once and Припев will not follow automatically. If Припев is needed after Вставка, please add it manual.
&Куплет.This is to be used only when Куплет is large and needs to be shown one more than one slide. &Куплет must follow Куплет otherwise there will be formating error. The &Куплет tells the program that it is part of Куплет that is above it.
&Припев.This is to be used only when Припев is large and needs to be shown one more than one slide. &Припев must follow Припев otherwise there will be formating error. The &Припев tells the program that it is part of Припев that is above it. The program will create Припев list that will be automatically repeated after each Куплет.


Example 1 - as Edited.Куплет1, Припев, Куплет2, Куплет3.
Example 1 - Shown.Куплет1, Припев, Куплет2,Припев, Куплет3, Припев.
Example 2 - as Edited.Куплет1, Припев, Куплет2, Вставка, Куплет3.
Example 2 - Shown.Куплет1, Припев, Куплет2,Припев, Вставка,Куплет3, Припев.
Example 3 - as Edited.Куплет1, Припев1, Куплет2, Куплет3, Припев2.
Example 3 - Shown.Куплет1, Припев1, Куплет2,Припев1, Куплет3, Припев2.
Example 4 - as Edited.Куплет1a, &Куплет1b, Припев, Куплет2a, &Куплет2b, Куплет3a, &Куплет3b.
Example 4 - Shown.Куплет1a, Куплет1b, Припев, Куплет2a, Куплет2b, Припев, Куплет3a, Куплет3b,Припев.
Example 5 - as Edited.Куплет1a, &Куплет1b, Припев-a, &Припев-b, Куплет2a, &Куплет2b, Куплет3a, &Куплет3b.
Example 5 - Shown.Куплет1a, Куплет1b, Припев-a, Припев-b, Куплет2a, Куплет2b, Припев-a, Припев-b, Куплет3a, Куплет3b, Припев-a, Припев-b.

To add a new songbook:
When adding a song and you have selected to add a songbook, a diffirent dialog will pop up where you will enter new songbooks information.
"Songbook title".Enter the title of the new songbook.
"Songbook information". Enter some information about new songbook.
Click "Ok" button.This will add a new songbook into the database.
The dialog box will close and now you will be able to add a new song.
